# 随手记 · 工作计划

一个**手机离线随身记录**工具：突发想法随手记下，自动按天归档成每日工作计划，支持勾选完成与删除。**纯前端、零服务器、数据存在手机本地**，断网也能用。

---

## ✨ 功能

- **随手记录**：底部输入框，回车或点「＋」立即添加
- **自动每日计划**：每条记录自动带日期，按「今天 / 昨天 / 某天」分组，当天的记录即当天工作计划
- **勾选完成**：点左侧圆圈，完成的条目自动划线 + 标记「已完成」
- **删除**：右侧「✕」删除单条（带确认）
- **筛选**：全部 / 今日 / 未完成 / 已完成
- **搜索**：实时模糊搜索
- **统计**：顶部显示总条数与完成数
- **备份**：导出 / 导入 JSON，方便换手机迁移
- **离线**：PWA 技术，添加到主屏后断网可用，数据存 localStorage
- **手机优化**：大字、大按钮、适配刘海屏与安全区

## 📱 在手机上使用（两种方案）

### 方案 A：部署到免费静态托管（推荐，可离线 + 添加主屏幕）

1. 把整个 `随手记录` 文件夹上传到任意静态托管：
   - **GitHub Pages**：新建仓库 → 上传文件 → Settings → Pages → 选 `main` 分支根目录
   - **Vercel / Netlify**：拖拽文件夹即可，自动生成网址
   - **Cloudflare Pages** 同理
2. 手机浏览器打开生成的网址
3. 点浏览器菜单 → **「添加到主屏幕」**（iOS Safari 分享按钮 → 添加到主屏幕）
4. 之后从主屏幕图标打开，即为**离线 App**，断网照常记录

> 为什么需要托管：Service Worker（离线核心）要求 https 或 localhost 环境，本地直接打开文件无法启用离线缓存。

### 方案 B：电脑本地直接打开（仅记录，无离线缓存）

- 双击 `index.html` 用浏览器打开即可使用（数据存浏览器本地）
- 适合先体验功能；但不会离线，换设备数据不互通

## 🔒 隐私

所有数据只存在**你当前设备的浏览器**里，不上传任何服务器。建议定期用「导出备份」保存一份 JSON 到手机或云盘。
